# GMAE - GuildQuest Mini-Adventure Environment

Group 18 | INF122 Final Project

## Mini-Adventures
- **Caravan Trade Run** — Race across realms to fulfill trade orders and hit a gold target
- **Relic Hunt** — Navigate a grid map to collect the most relics before rounds run out

## Requirements
- Java 17+
- Maven 3.6+

## How to Run
```bash
mvn javafx:run
```
